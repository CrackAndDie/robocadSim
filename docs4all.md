# Docs for all versions of robocadSim  
<h3> <strong>Keys:</strong> </h3>  
<ul>
  <li> <strong>Esc:</strong> open "pause" menu </li>  
  <li> <strong>LeftCtrl + E:</strong> open power panel </li>  
  <li> <strong>LeftCtrl + D:</strong> open control panel </li>  
</ul>
